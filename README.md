# Dashboard RD Construtora

Dashboard de faturamento profissional para a RD Construtora, desenvolvido com HTML5, CSS3 e JavaScript puro.

## 🎯 Funcionalidades

- **Resumo Anual**: Visualização de faturamento por mês
- **KPIs em Tempo Real**: Faturamento anual, mensal e meta de vendas
- **Gráficos Interativos**: Evolução mensal com Chart.js
- **Gestores**: Resumo de performance dos gestores
- **Atualização de Dados**: Modal para editar todos os dados em tempo real
- **Relógio Digital**: Exibição de hora e data em tempo real
- **Design Profissional**: Interface moderna com tema escuro

## 🚀 Deploy

Este projeto está hospedado na Vercel e pode ser acessado em:
[https://dashboard-rd-construtora.vercel.app](https://dashboard-rd-construtora.vercel.app)

## 📝 Uso

1. Acesse o dashboard
2. Clique no botão **"✏ ATUALIZAR DADOS"** para editar informações
3. Atualize a meta mensal, mês atual, faturamento e gestores
4. Clique em **"APLICAR"** para salvar as alterações

## 🛠️ Tecnologias

- HTML5
- CSS3 (Grid, Flexbox, Animações)
- JavaScript Vanilla
- Chart.js 4.4.1
